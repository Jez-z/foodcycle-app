# Tech Stack FoodCycle App

## 1. React.js

React.js digunakan sebagai library utama untuk membangun tampilan frontend aplikasi FoodCycle App. React memudahkan pembuatan tampilan berbasis komponen, seperti Navbar, halaman Home, daftar produk, detail produk, tambah produk, dan checkout.

Dengan React, setiap bagian tampilan dapat dibuat menjadi komponen terpisah sehingga kode lebih rapi, mudah digunakan ulang, dan lebih mudah dikembangkan oleh anggota tim.

## 2. Vite

Vite digunakan sebagai build tool untuk menjalankan dan mengembangkan aplikasi React. Vite dipilih karena proses development-nya lebih cepat dan ringan.

Dengan Vite, aplikasi dapat dijalankan secara lokal menggunakan perintah:

'npm run dev'

Hal ini membantu developer melihat perubahan tampilan aplikasi secara langsung saat proses pengembangan.

## 3. JavaScript

JavaScript digunakan sebagai bahasa utama dalam pengembangan aplikasi. JavaScript berfungsi untuk mengatur logic aplikasi, seperti perpindahan halaman, pengelolaan data produk, interaksi user, dan proses sederhana seperti checkout.

## 4. CSS

CSS digunakan untuk mengatur tampilan aplikasi agar lebih menarik dan mudah digunakan. CSS berperan dalam mengatur warna, layout, tombol, card produk, jarak antar elemen, dan tampilan responsive.

## 5. HTML

HTML digunakan sebagai struktur dasar halaman web. Pada project berbasis Vite dan React, struktur utama halaman terdapat pada file index.html, lalu tampilan aplikasi dirender melalui React.

## 6. React Router

React Router digunakan untuk mengatur navigasi antar halaman dalam aplikasi. Dengan React Router, user dapat berpindah dari halaman Home ke halaman daftar produk, detail produk, tambah produk, dan checkout tanpa perlu reload halaman secara penuh.

## 7. GitHub

GitHub digunakan sebagai tempat penyimpanan source code dan version control. Dengan GitHub, anggota kelompok dapat bekerja secara kolaboratif, menyimpan perubahan melalui commit, dan menggabungkan hasil kerja ke repository utama.

## Kesimpulan

Tech stack yang digunakan dalam FoodCycle App berfokus pada pengembangan frontend. React.js digunakan untuk membangun tampilan aplikasi, Vite untuk mempercepat proses development, JavaScript untuk logic aplikasi, CSS untuk styling, HTML untuk struktur dasar, React Router untuk navigasi, dan GitHub untuk kolaborasi tim.
